# SECOM 반도체 공정 불량 원인 분석

## 프로젝트 개요
UCI SECOM 데이터셋을 활용해 591개 공정 센서 중 불량(Fail)에 영향을 주는 핵심 변수를 통계적으로 추적

## 데이터
- 출처: UCI Machine Learning Repository
- 구성: 1567 샘플 x 591 센서 + Pass/Fail 라벨
- 불량 비율: 6.6% (Fail 104개)

## 분석 파이프라인
1. 데이터 로드 및 기본 탐색
2. 결측치 처리 (50% 초과 컬럼 제거 + 평균 대체)
3. 라벨 불균형 시각화
4. t-검정으로 유의미한 센서 추출 (564개 -> 79개)
5. 센서 간 상관관계 분석
6. 로지스틱 회귀 모델 (Fail Recall: 0.38)
7. 핵심 센서 중요도 해석

## 주요 결과
- 센서 121번이 Fail과 가장 강한 양의 상관관계
- 핵심 센서 모니터링으로 불량 조기 감지 가능성 확인
